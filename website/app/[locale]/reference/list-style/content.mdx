## Conditionally apply
Apply styles based on different states using [selectors](/guide/state-selectors) and [applying conditionally](/guide/applying-conditionally).
```html
<div class="list-style:url(…)|inside**:hover** list-style:url(…)|inside**@sm** list-style:url(…)|inside**@dark** list-style:url(…)|inside**@print**">…</div>
```
