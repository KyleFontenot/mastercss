## Setup
### Create a config file
Initialize Master CSS in an existing project.
<CodeTabs showControls>
{[
    {
        name: 'npm', lang: 'bash',
        code: 'npm create @master/css@rc',
    },
    {
        name: 'pnpm', lang: 'bash',
        code: 'pnpm create @master/css@rc',
    },
    {
        name: 'yarn', lang: 'bash',
        code: 'yarn create @master/css@rc',
    }
]}
</CodeTabs>

### Create a new app
Or you can specify an argument to create a new project, which by default uses the [blank example](https://github.com/master-co/css/tree/rc/examples/blank).
<CodeTabs showControls>
{[
    {
        name: 'npm', lang: 'bash',
        code: 'npm create @master/css@rc __project__',
    },
    {
        name: 'pnpm', lang: 'bash',
        code: 'pnpm create @master/css@rc __project__',
    },
    {
        name: 'yarn', lang: 'bash',
        code: 'yarn create @master/css@rc __project__',
    }
]}
</CodeTabs>

## Interfaces
### `create --ext esm`
Specify creating a `master.css.mjs` file in ECMAScript Modules format. If `package.json` is set `"type": "module"` will be enabled automatically.
<DocProp defaultValue="auto" types={['boolean']} />
```bash name=Terminal
npm create @master/css@rc **--ext esm**
```

    <summary>View the generated file</summary>

    </div>
</details>

### `create --example`
Create a Master CSS App using [official examples](https://github.com/master-co/css/tree/rc/examples).
<DocProp defaultValue="'blank'" types={['string']} />
```bash name=Terminal
npm create @master/css@rc __project__ **--example react**
```

### `create --pm`
Manually specify the package manager you are using.
<DocProp defaultValue="auto" types={['string']} />
```bash name=Terminal
npm create @master/css@rc **--pm bun**
```

### `create -o, --override`
Override the existing Master CSS config file.
<DocProp defaultValue="false" types={['boolean']} />
```bash name=Terminal
npm create @master/css@rc **--override**
```
