## Overview [sr-only]
<Overview/>

### Positioning elements as static
Use `static`to position an element according to the normal flow of the document. All HTML elements are positioned static by default.
<Demo className="pb:60">
    <code className="static p:20 box-perspective-pink">
        <div>static parent</div>
        <code className="abs left:10 bottom:10 box w:fit">abs child</code>
    </code>
</Demo>
```html
<div class="**static** p:20">
    <div>static parent</div>
    <div class="abs left:10 bottom:10">abs child</div>
</div>
```

### Positioning elements as relative
Use `rel`to position an element according to the normal flow of the document, and then offset relative to itself based on the values of top, right, bottom, and left.
<Demo className="pb:60">
    <div className="bg:stripe">
        <code className="rel p:20 top:30 left:20 box-perspective-pink inline-block">
            <div>relative parent</div>
        </code>
    </div>
</Demo>
```html
<div class="**rel** p:20 top:30 left:20">
    <div>relative parent</div>
</div>
```

### Positioning elements as absolute
Use `abs`to remove an element from the normal flow of the document, and the element positioned relative to its closest positioned ancestor.
<Demo>
    <code className="rel p:12x! box bg:stripe">
        <div>relative parent</div>
        <code className="box-perspective-pink abs left:0 top:0 w:fit">abs child</code>
    </code>
</Demo>
```html
<div class="**rel** p:12x">
    <div>relative parent</div>
    <div class="**abs** left:0 top:0">abs child</div>
</div>
```

### Positioning elements as fixed
Use `fixed`to remove an element from the normal flow of the document. The element positioned relative to the viewport, and it is always stays in the same place.
<Demo>
    <div className="box ai:initial! jc:initial! p:8|20_li p:10|0! r:10 w:full max-w:340@sm max-h:250 overflow:auto">
        <input className="fixed z:1 bg:gray-23 bg:gray-18:focus fg:gray-40::placeholder outline:0 font:14 w:300 mx:10 p:12 leading:1 r:5" placeholder="Search for chats and messages" type="search" />
        <ul className="list-style:none mt:10 pl:0 w:full">
            {users.map((user, index) =>
                <li className="bg:gray-28:hover bg:gray-32:active flex gap:12 cursor:pointer" key={index}>
                    <div className="rel">
                        <Image src={user.url} width="40" height="40" className="round object:cover" alt="avatar" />
                        {index < 2 && <svg className="abs right:1 bottom:1 bg:green-70 outline:2|solid|white outline:gray-20@dark round w:7 h:7" />}
                    </div>
                    <div>
                        <div className="flex align-items:center">
                            <div className="text:14 font:medium fg:gray-88@dark capitalize">{user.name}</div>
                            <div className="font:12 fg:slate-54 fg:gray-44@dark ml:8">{user.time}</div>
                        </div>
                        <p className="text:13 m:0 t:left">{user.message}</p>
                    </div>
                </li>
            )}
        </ul>
    </div>
</Demo>
```html
<div>
    <input type="search" class="**fixed**">
    <ul>

        …
    </ul>
</div>
```

### Positioning elements as sticky
Use `sticky`to position an element base on scroll position. It&#39;s like place the element to a relative position until a given offset position is met and then changing the element to a fixed position.
<Demo>
    <div className="box block! p:8|20_li p:10|0! r:10 w:full max-w:340@sm max-h:250 overflow:auto">
        <ul className="list-style:none mt:10 pl:0 w:full">
            {users.map((user, index) =>
                <li className={clsx(index === 1 ? 'sticky top:0 z:1' : '', 'bg:gray-30 bg:gray-28:hover bg:gray-32:active flex gap:12 cursor:pointer')} key={index}>
                    <div className="rel">
                        <Image src={user.url} width="40" height="40" className="round object:cover" alt="avatar" />
                        {index < 2 && <svg className="abs right:1 bottom:1 bg:green-70 outline:2|solid|white outline:gray-20@dark round w:7 h:7" />}
                    </div>
                    <div>
                        <div className="flex align-items:center">
                            <div className="text:14 font:medium fg:gray-88@dark capitalize">{user.name}</div>
                            <div className="font:12 fg:slate-54 fg:gray-44@dark ml:8">{user.time}</div>
                        </div>
                        <p className="text:13 m:0 t:left">{user.message}</p>
                    </div>
                </li>
            )}
        </ul>
    </div>
</Demo>
```html
<ul class="overflow:auto">
    …
    <li class="**sticky** top:0">
        <img src="…">
        <div>Joy</div>
        …
    </li>
    …
</ul>
```

## Conditionally apply
Apply styles based on different states using [selectors](/guide/state-selectors) and [conditional queries](/guide/conditional-queries).
```html
<div class="rel**:hover** rel**@sm** rel**@dark** rel**@print**">…</div>
```
