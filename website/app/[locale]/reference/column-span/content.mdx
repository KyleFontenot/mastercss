## Basic usage
### Span across all columns
Use `col-span:all` to make elements span across all columns.
<Basic className="col-span:all" />

### Does not span multiple columns
Use `col-span:none` to make elements not span across multiple columns.
<Basic className="col-span:none" />

---

## Conditionally apply
Apply styles based on different states using [selectors](/guide/state-selectors) and [conditional queries](/guide/conditional-queries).
```html
<div class="col-span:all**:hover** col-span:all**@sm** col-span:all**@dark** col-span:all**@print**">â€¦</div>
```
