## Basic usage
### Apply a base color
By default, all base colors are set at the top level of variables and accessed as a global variable.
```html
<html class="**light**">
-    <body class="bg:$(base)">equals</body>
+    <body class="bg:base"></body>
</html>
```

---

## Customization
### Add a base color
Add a color to the base variables group.
```js name=master.css.js
export default {
    variables: {
+        primary: '#000'
    }
}
```
Apply the custom color:
```html
<p class="bg:**primary**"></p>
```
See the [Color Customization](/docs/colors#customization) documentation to learn more about color variable behavior.
