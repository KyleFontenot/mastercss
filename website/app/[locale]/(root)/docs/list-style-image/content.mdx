## Conditionally apply
Apply styles based on different states using [selectors](/docs/state-selectors) and [conditional queries](/docs/conditional-queries).
```html
<div class="list-style-image:url(…)**:hover** list-style-image:url(…)**@sm** list-style-image:url(…)**@dark** list-style-image:url(…)**@print**">…</div>
```
