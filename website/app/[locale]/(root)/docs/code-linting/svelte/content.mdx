<StepSection className="mt:12x">
    <Step $row>
        <StepL>
            ### <StepNum />Installation
            Install the plugin and parser into your project via package managers.
        </StepL>
        <StepR>
            ```install
            -D @master/eslint-config-css@beta eslint-plugin-svelte eslint
            ```
        </StepR>
    </Step>
    <Step $row>
        <StepL>
            ### <StepNum />Set up your config
            Create [`.eslintrc.json`](https://eslint.org/docs/latest/use/getting-started) if you don't have one.
            - Add `@master/css` to the `extends` section.
            - Add `plugin:svelte/recommended` to the `extends` section.
        </StepL>
        <StepR>
            ```json name=.eslintrc.json
            {
                "extends": [
            +         "@master/css",
            +         "plugin:svelte/recommended"
                ]
            }
            ```
        </StepR>
    </Step>
    <Step $row>
        <StepL>
            ### <StepNum />Add VS Code settings
            Add the `eslint.validate` option of the extension to validate `.svelte` files because the extension targets only `*.js` or `*.jsx` files by default.

            Check out the [Visual Studio Code guide](/docs/code-linting/vscode) if you haven't installed the ESLint extension yet.
        </StepL>
        <StepR>
            ```json name=.vscode/settings.json
            {
                "eslint.validate": [
                    "javascript",
                    "javascriptreact",
            +        "svelte"
                ]
            }
            ```
        </StepR>
        <StepEnd/>
    </Step>
    <Step>
        ### <StepNum />Start linting your code
        ```bash
        npx eslint *.svelte
        ```
    </Step>
</StepSection>
