## Basic usage
### Set the cursor style
Use the `cursor:value` to change the mouse cursor when hovering an element.

(hover) Hovering the button to see the cursor style
<Demo className="gap:40">
    <button className="fg:white bg:indigo px:20 h:42 font:14 font:semibold r:5 cursor:pointer">submit</button>
</Demo>
```html
<button class="**cursor:pointer**">submit</button>
```

---

## Conditionally apply
Apply styles based on different states using [selectors](/docs/state-selectors) and [conditional queries](/docs/conditional-queries).
```html
<div class="cursor:pointer**:hover** cursor:pointer**@sm** cursor:pointer**@dark** cursor:pointer**@print**">â€¦</div>
```
