## Basic usage
### Add a color
Add color variables using hex codes and extend the defaults.
```js name=master.css.js
export default {
    variables: {
+        primary: '#0e61fe'
    }
}
```
Apply them using color-related syntax like [`fg:`](/docs/color).
```html
<div class="**fg:primary**">…</div>
```

### Add an alias
Create an alias for a color to link its value to an existing color. This allows you to implement [design tokens](/docs/design-tokens) and makes managing design updates more efficient.
```js name=master.css.js
export default {
    variables: {
+        secondary: '$(blue-60)'
    }
}
```
For example, say you have multiple color variables referencing the same color token. If that color needs updating, you would only need to update the source instead of manually updating every instance of the color.

This is similar to [Figma's variables](https://help.figma.com/hc/en-us/articles/15145852043927-Create-and-manage-variables).

### Add color shades
Sets multiple shades for a single color.
```js name=master.css.js
export default {
    variables: {
+        primary: {
+            '': '#1192e8', // primary
+            10: '#e5f6ff', // primary-10
+            20: '#bae6ff', // primary-20
+        }
    }
}
```
Apply them using color-related syntax like [`fill:`](/docs/fill).
```html
<svg class="**fill:primary-20** …">20</svg>
```

### Override default colors
(!) Access the same key as the preset to override the default color.
```js name=master.css.js
export default {
    variables: {
+        blue: '#4589ff'
    }
}
```
(o) Use `app-*` to avoid conflicts with defaults and make Master colors always consistent.
```js name=master.css.js
export default {
    variables: {
+        app: {
+            blue: '#4589ff' // app-blue
+        }
    }
}
```

## Advanced usage
### Add colors by theme
Add color variables based on different themes.
```js name=master.css.js
export default {
    variables: {
+        primary: {
+            '@light': '$(yellow-40)',
+            '@dark': '$(yellow-50)'
+        }
    }
}
```
Set the theme, `light` or `dark`, to apply color variables.
```html
<div class="**light**">
    <div class="bg:**primary**">…</div>
</div>
```

## Custom references
### Using text color tokens
| Token             | Description                                                                                                                       |
|-------------------|-----------------------------------------------------------------------------------------------------------------------------------|
| `text-accent`     | Accent color emphasizes active links and states, strongly contrasting the background.                                             |
| `text-strong`     | Strong color, used for main titles, subheadings, or slightly emphasized fields, creating a high contrast with the background.     |
| `text-neutral`    | Soft color, a foreground color harmonizing with the background.                                                                   |
| `text-light`      | Light color, a foreground color contrasting slightly with the background.                                                         |
| `text-lighter`    | Lighter color, a foreground color contrasting slightly with the background.                                                       |
| `text-lightest`   | Dim color, a foreground color contrasting slightly with the background.                                                           |

Create foreground color variables for titles, descriptions, icons, and other body text.
```js name=master.css.js
export default {
    variables: {
        text: {
+            primary: '$(blue-60)'
        }
    }
}
```
Apply the defined `text-*` variables using the inherited rules <TextHeirs />.
```html
<h1 class="**fg:primary**">page title</h1>
```

### Using surface color tokens
| Token             | Description                                                                                                                       |
|-------------------|-----------------------------------------------------------------------------------------------------------------------------------|
| `base`            | The background color for apps, modals, ...                                                                                        |

Create color variables for backgrounds in apps, panels, cards, and more.
```js name=master.css.js
export default {
    variables: {
+        base: '$(white)'
    }
}
```
Use [background-color](/docs/background-color) syntax to set the app background color:
```html
<body class="**bg:base**">…</body>
```
