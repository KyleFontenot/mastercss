## Basic usage
### Color tokens
Change the text color of an element using the default [color variables](/docs/colors).
<Demo>
    <div className="fg:blue font:56 font:heavy">Aa</div>
</Demo>
```html
<div class="**fg:blue**">Aa</div>
```

---

## Conditionally apply
Apply styles based on different states using [selectors](/docs/state-selectors) and [conditional queries](/docs/conditional-queries).
```html
<div class="fg:slate-10**:hover** fg:slate-10**@sm** fg:slate-10**@dark** fg:slate-10**@print**">â€¦</div>
```
