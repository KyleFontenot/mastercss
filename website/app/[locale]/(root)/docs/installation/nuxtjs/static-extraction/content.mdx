## Quick start
<StepSection>
    <Step $row>
        <StepL>
            ### <StepNum />Clone the example
            Copy-paste the following commands to quickly start using the [nuxtjs.se.rc.css.master.co](https://nuxtjs.se.rc.css.master.co).

            You can skip all [installation](#installation) steps.
        </StepL>
        <StepR>
            ```bash name=Terminal
            npx degit master-co/css/examples/nuxt.js-with-static-extraction __my-project__
            cd __my-project__
            npm install @master/css-extractor.vite@rc
            npm install
            npm run dev
            ```
        </StepR>
        <StepEnd/>
    </Step>
</StepSection>

## Installation

<StepSection>
    <Step $row>
        <StepL>
            ### <StepNum />Create a project
            If you don't have a Nuxt.js project, create one first. It's recommended to refer to [Get started with Nuxt.js](https://nuxt.com/docs/getting-started/installation).
        </StepL>
        <StepR>
            ```bash name=Terminal
            npx nuxi@latest init __project__
            cd __project__
            ```
        </StepR>
    </Step>
    <Step $row>
        <StepL>
            ### <StepNum />Install Master CSS
            Install the Master CSS extractor into your project via package managers.
        </StepL>
        <StepR>
            ```install
            **@master/css-extractor.vite@rc**
            ```
        </StepR>
    </Step>
    <Step $row>
        <StepL>
            ### <StepNum />Initialize configuration file
            Run `npx @master/css init` to create a configuration file [master.css.ts](/docs/configuration).
        </StepL>
        <StepR>
            ```bash name=Terminal
            npx @master/css init
            ```
        </StepR>
    </Step>
    <Step $row>
        <StepL>
            ### <StepNum />Set up CSS extractor
            Add a `CSSExtractorPlugin` to vite plugins in `nuxt.config.ts`.
        </StepL>
        <StepR>
            ```js name=nuxt.config.ts
            +import { CSSExtractorPlugin } from '@master/css-extractor.vite'

            export default defineNuxtConfig({
            +    vite: {
            +        plugins: [
            +            CSSExtractorPlugin()
            +        ]
            +    }
            })
            ```
        </StepR>
    </Step>
    <Step $row>
        <StepL>
            ### <StepNum />Import virtual CSS module
            Import the virtual CSS module `.virtual/master.css` into the `app.vue`.
        </StepL>
        <StepR>
            ```html name=app.vue
            <script setup lang="ts">
            +    import ".virtual/master.css";
            </script>
            …
            ```
        </StepR>
    </Step>
    <Step $row>
        <StepL>
            ### <StepNum />Launch server
            Run `npm run dev` to start your Nuxt.js development server
        </StepL>
        <StepR>
            ```bash name=Terminal
            npm run dev
            ```
        </StepR>
    </Step>
    <Step>
        ### <StepNum />Start using Master CSS
        Now style your first element using Master CSS syntax!
        ```html name=app.vue
        …
        <template>
            <h1 class="**font:40** **font:heavy** **italic** **m:12x** **text:center**">Hello World</h1>
        </template>
        ```
        <p className="mt:12x">Open your browser to watch the changes.</p>
        <HelloWorld url="localhost:3000" />
    </Step>
</StepSection>
