## Quick start
<StepSection>
    <Step $row>
        <StepL>
            ### <StepNum />Clone the example
            Copy-paste the commands to quickly start using the https://svelte.se.rc.css.master.co example.

            You can skip all [installation](#installation) steps.
        </StepL>
        <StepR>
            ```bash name=Terminal
            npm create @master/css@rc __project__ --example svelte-with-static-extraction
            cd __project__
            npm run dev
            ```
        </StepR>
        <StepEnd/>
    </Step>
</StepSection>

## Installation

<StepSection>
    <Step $row>
        <StepL>
            ### <StepNum />Create a project
            If you don't have a Svelte project, create one first. It's recommended to refer to [Svelte Kit](https://kit.svelte.dev/).
        </StepL>
        <StepR>
            ```bash name=Terminal
            npm create svelte@latest __project__
            cd __project__
            npm install
            ```
        </StepR>
    </Step>
    <Step $row>
        <StepL>
            ### <StepNum />Initialize configuration file
            Run `npm create @master/css@rc` to create a configuration file [master.css.ts](/reference/configuration).
        </StepL>
        <StepR>
            ```bash name=Terminal
            npm create @master/css@rc
            ```
        </StepR>
    </Step>
    <Step $row>
        <StepL>
            ### <StepNum />Install Master CSS
            Install the Master CSS extractor into your project via package managers.
        </StepL>
        <StepR>
            ```install
            **@master/css-extractor.vite@rc**
            ```
        </StepR>
    </Step>
    <Step $row>
        <StepL>
            ### <StepNum />Set up CSS extractor
            - Create a `vite.config.ts` file
            - Add a `CSSExtractorPlugin` to the vite `plugins`
        </StepL>
        <StepR>
            ```ts name=vite.config.ts
            +import { defineConfig } from 'vite'
            +import CSSExtractorPlugin from '@master/css-extractor.vite'

            +export default defineConfig({
            +    plugins: [
            +        CSSExtractorPlugin()
            +    ]
            +})
            ```
        </StepR>
    </Step>
    <Step $row>
        <StepL>
            ### <StepNum />Import virtual CSS module
            Import the virtual CSS module `.virtual/master.css` into the `src/routes/+layout.svelte`.
        </StepL>
        <StepR>
            ```svelte name=src/routes/%2Blayout.svelte
            <script>
                import Header from './Header.svelte';
                import './styles.css';
            +    import '.virtual/master.css';
            </script>
            â€¦
            ```
        </StepR>
    </Step>
    <Step $row>
        <StepL>
            ### <StepNum />Launch server
            Run `npm run dev -- --open` to start your Vite development server
        </StepL>
        <StepR>
            ```bash name=Terminal
            npm run dev -- --open
            ```
        </StepR>
    </Step>
    <Step>
        ### <StepNum />Start using Master CSS
        Now style your first element using Master CSS syntax!
        ```svelte name=src/routes/%2Bpage.svelte
        +<h1 class="font:40 fg:blue font:heavy italic m:12x text:center">Hello World</h1>
        ```
        <p className="mt:12x">Open your browser to watch the changes.</p>
        <HelloWorld url="localhost:5173" />
    </Step>
</StepSection>
