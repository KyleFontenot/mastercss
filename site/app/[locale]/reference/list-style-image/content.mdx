## Conditionally apply
Apply styles based on different states using [selectors](/guide/state-selectors) and [applying conditionally](/guide/applying-conditionally).
```html
<div class="list-style-image:url(…)**:hover** list-style-image:url(…)**@sm** list-style-image:url(…)**@dark** list-style-image:url(…)**@print**">…</div>
```
