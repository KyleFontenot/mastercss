## Features [sr-only]
```html
<!-- Set foreground to blue on hover -->
<div class="fg:blue**:hover**">

<!-- Set border to red on input invalid -->
<input class="border:red**:invalid**" />

<!-- Set background to white on expanded -->
<details class="bg:white**[open]**" open>
…
```
Master CSS allows you to suffix, such as [pseudo-classes](#pseudo-classes), [pseudo-elements](#pseudo-elements), [compound states](#compound-states), etc., to apply styles on different conditions.

### Multiple states
Master CSS allows the same property to have different states on the same element.
```html
<h2 class="**font:18** **font:24@sm** **font:32@md** …">
```

### Stackable states
Master CSS allows the same style declaration with multiple state conditions.
```html
<div class="**bg:blue-20:hover@md@dark** …">
```

## Pseudo-classes
### Hover, active, focus
Style elements with different interactive appearances on `:hover`, `:active`, and `:focus`.

(click) Try clicking the button to see states change
<Demo>
    <button className="bg:indigo bg:pink:hover bg:purple:active px:18 h:40 font:14 font:semibold fg:white r:4 inline-flex align-items:center">Submit</button>
</Demo>
```html
<button class="bg:indigo bg:pink**:hover** bg:purple**:active**">Submit</button>
```

### First, last, even
Select and style elements using `:first`, `:last`, `:odd`, `:even`, and `:nth(N)` shorthands.
<Demo $py={0}>
    <DemoPanel $p={0} className="w:full max-w:360">
        <ul className="list-style:none p:20|0 m:0">
            {furnitures.map((furniture, index) =>
                <li className="bb:0:last bg:slate-5:even@light bg:gray-90:even@dark bb:1|lightest px:25 flex align-items:center my:0 gap:12" key={index}>
                    <img className="round object:cover" src={furniture.url} width="56" height="56" alt="avatar" quality={'0'} />
                    <div className="flex flex:col flex:1 py:10">
                        <div className="flex align-items:center">
                            <div className="text:14 capitalize fg:strong">{furniture.name}</div>
                        </div>
                        <div className="text:12 m:0 fg:neutral">{furniture.intro}</div>
                    </div>
                </li>
            )}
        </ul>
    </DemoPanel>
</Demo>
```html
<ul class="…">
    <li class="bg:slate-5**:even**@light bg:gray-90**:even**@dark …">…</li>
</ul>
```

### Form validations
Style controls using `:valid`, `:invalid`, `:required` based on form validations.

(type) Try typing `co` as a valid email address
<Demo>
    <input type="email" className="b:red:invalid b:green:valid b:1|lightest outline:0 h:48 font:14 px:16 bg:panel r:5 max-w:300" defaultValue="aron@master." />
</Demo>
```html
<input class="b:red**:invalid** b:green**:valid** b:1|solid …" type="email" />
```

## Pseudo-elements
### Before, after
Attach pseudo-elements to an element using `::before` and `::after`.
<Demo $py={0}>
    <DemoPanel>
        <q className="italic fg:strong font:medium font:18">Be the change you wish to see in the world.</q>
    </DemoPanel>
</Demo>
```html
<div class="content:open-quote**::before** content:close-quote**::after** …">
    Be the change you wish to see in the world.
</div>
```

### Scrollbar
Beautify the scrollbar of an element using `::scrollbar`, `::scrollbar-thumb`, `::scrollbar-track`, `::scrollbar-button`, `::scrollbar-corner` and `::scrollbar-track-piece` shorthand.

(hover) Try scrolling horizontally and interact with the scroll bar
<Demo $py={0} $px={0}>
    <div className="size:6::scrollbar bg:gray/.2::scrollbar-thumb overflow-x:auto p:40">
        <div className='flex w:max'>
            {animals.map((animal) =>
                <div key={animal.name}>
                    <Image src={animal.url} width={128} height={128} alt={animal.name} />
                </div>
            )}
        </div>
    </div>
</Demo>
```html
<div class="size:6**::scrollbar** bg:gray/.2**::scrollbar-thumb** overflow-x:auto …">
```
