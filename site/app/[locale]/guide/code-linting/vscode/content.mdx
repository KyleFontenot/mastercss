    <Step $row>
        <StepL>
            ### <StepNum />Install Master CSS ESLint
            Install the Master CSS ESLint Config into your project via package managers.
        </StepL>
        <StepR>
            ```install
            -D @master/eslint-config-css@rc eslint
            ```
        </StepR>
    </Step>
    <Step $row>
        <StepL>
            ### <StepNum />Set up ESLint config
            Create [`.eslintrc.json`](https://eslint.org/docs/latest/use/getting-started) if you don't have one.
            - Add `@master/css` to the `extends` section.
        </StepL>
        <StepR>
            ```json name=.eslintrc.json
            {
                "extends": ["@master/css"]
            }
            ```
        </StepR>
    </Step>
    <Step $row>
        <StepL>
            ### <StepNum />Add VS Code settings
            Create a `.vscode/settings.json` or set it in [user settings](https://code.visualstudio.com/docs/getstarted/settings).

            - Add `@master/css/class-order` to the `eslint.codeActionsOnSave.rules` section.
            - Add the language you want to lint to `eslint.validate`. The VSCode ESLint targets only `javascript` or `javascriptreact` files by default.
        </StepL>
        <StepR>
            ```json name=.vscode/settings.json
            {
                "eslint.codeActionsOnSave.rules": [
                    "@master/css/class-order"
                ],
                "eslint.validate": [
                    "html",
                    "php",
                    "javascript",
                    "typescript",
                    "javascriptreact",
                    "typescriptreact",
                    "vue",
                    "svelte",
                    "rust",
                    "astro",
                    "markdown",
                    "mdx"
                ]
            }
            ```
        </StepR>
        <StepEnd/>
    </Step>
</StepSection>
