        <StepL>
            ### <StepNum />Installation
            Install the plugin and parser into your project via package managers.
        </StepL>
        <StepR>
            ```install
            -D @master/eslint-config-css@rc eslint-plugin-vue @vue/eslint-config-typescript eslint
            ```
        </StepR>
    </Step>
    <Step $row>
        <StepL>
            ### <StepNum />Set up ESLint config
            Import and add the Master CSS ESLint Configuration `html css` to [`eslint.config.mjs`](https://eslint.org/docs/latest/use/getting-started).

            It is recommended to refer to the [official guide](https://eslint.vuejs.org).
        </StepL>
        <StepR>
            ```js name=eslint.config.mjs
            import vue from 'eslint-plugin-vue'
            import vueTS from '@vue/eslint-config-typescript'
            import css from '@master/eslint-config-css'

            export default [
                { files: ['**/*.{ts,mts,tsx,vue}'] },
                { ignores: ['**/dist/**', '**/dist-ssr/**', '**/coverage/**'] },
                ...vue.configs['flat/essential'],
                ...vueTS(),
                css
            ]
            ```
        </StepR>
    </Step>
    <Step>
        ### <StepNum />Start linting your code
        ```bash
        npx eslint
        ```
    </Step>
</StepSection>
