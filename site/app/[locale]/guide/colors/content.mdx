### Refer to default colors [sr-only]
<ColorPalette />
Each color in the [Master Colors](https://github.com/master-co/colors) has a hue range from 5 to 95, which you can choose from as your brand's primary color.

For example, inherit from indigo:
```js name=master.css.js
import { variables } from '@master/css'

/** @type {import('@master/css').Config} */
export default {
    variables: {
         primary: variables.indigo
    }
}
```
Apply them using the relevant syntax:
```html
<div class="**bg:primary-5** **fg:primary-60**">…</div>
```

### Add color shades
Sets multiple shades for a single color.
```js name=master.css.js
export default {
    variables: {
+        primary: {
+            '': '#1192e8', // primary
+            10: '#e5f6ff', // primary-10
+            20: '#bae6ff', // primary-20
+        }
    }
}
```
Apply it using color-related syntax like [`fill:`](/reference/fill).
```html
<svg class="**fill:primary-20** …">20</svg>
```

### Add a base color
Add a color to the base variables group.
```js name=master.css.js
export default {
    variables: {
+        primary: '#000'
    }
}
```
Apply the custom color:
```html
<p class="bg:**primary**"></p>
```

### Apply a base color
By default, all base colors are set at the top level of variables and accessed as a global variable.
```html
-<body class="bg:$(base)">equals</body>
+<body class="bg:base"></body>
```

### Add a text color
Add a color to the text variables group.
```js name=master.css.js
export default {
    variables: {
        text: {
+            primary: '#000'
        }
    }
}
```
Apply the custom color:
```html
<p class="fg:**primary**"></p>
```

### Apply a text color
Apply the defined `text-*` variables using the inherited syntaxes <TextHeirs />.
```html
<html class="**light**">
-    <body class="fg:$(text-neutral)">equals</body>
-    <body class="fg:text-neutral">equals</body>
+    <body class="fg:neutral"></body>
</html>
```
(!) Typically, we set the [foreground color](/reference/color) of elements using `fg:color` rather than the [text-specific](/reference/text-fill-color) `text:color`.
```html
-<p class="text:pink">text-fill-color properity</p>
+<p class="fg:pink">color properity</p>
```

### Apply text color to any syntax
Besides inheriting the above rules, you can use `text-*` to access global text color variables in other syntax.
```html
<svg class="stroke:**text-light**"></svg>
```

### Escape inherited text colors
Use `$(blue)` to apply `variables.blue` instead of `variables.text.blue`.
```html
<p class="fg:**$(blue)**">…</p>
```
