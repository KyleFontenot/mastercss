Animation

Style syntax for applying an animation between styles.

class properties
animation:name|duration|… / @name|duration|… animation: name duration …

Official animations

We provide a set of animations commonly used:

@fade|1s|infinite
@ping|1s|infinite
@flash|1s|infinite
@heart|1s|infinite
@jump|1s|infinite
@pulse|1s|infinite
@rotate|1s|infinite
@shake|1s|infinite
@zoom|1s|infinite
@float|3s|ease-in-out|infinite

Setup

1. Installation

npm install @master/animations.css

2. Import

@import '@master/animations.css';

or import only your demand

@import '@master/animations.css/fade';

or import via CDN

<link href="https://unpkg.com/@master/animations.css" rel="stylesheet">
<!-- or import only your demand -->
<link href="https://unpkg.com/@master/animations.css/fade.css" rel="stylesheet">

Basic usage

Add with custom animations

First, define @keyframes  in your css file or <style>. 

@keyframes demo {
    from {
        left: 0px;
        top: 0px;
        background: rgba(83, 140, 238, .2);
        border-color: rgb(83, 140, 238);
    }

    to {
        left: 150px;
        top: 0px;
        background: rgba(235, 95, 99, .5);
        border-color: rgb(235, 95, 99);
    }
}

Now, set the animation classes for the element.

Conditionally apply

Apply styles in different states using selectors, media queries, themes, and more syntax.

<div class="animation:fade:hover animation:fade@sm animation:fade@dark animation:fade@print">…</div>
